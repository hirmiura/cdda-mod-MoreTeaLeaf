茶葉を育てるMOD

説明

茶葉の種、生の茶葉、幾つかのお茶の葉とお茶を追加するMODです

茶葉の種は園芸店で入手できます

茶葉の種を育てると生の茶葉を収穫できます

制作にて紅茶の葉や緑茶、烏龍茶の葉にすることが出来ます

植木鉢MODに対応するパッチを既存のワールドに追加する場合はsaveフォルダの中のmods.jsonに mtl for Uekibati を追加して下さい
※植木鉢MODと茶葉を育てるMODが前提になるのでないとエラーが出ると思います
